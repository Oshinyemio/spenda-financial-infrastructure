# Spenda API Documentation

This directory contains the OpenAPI 3.0 specification for the Spenda backend. 

## Key Features
- **Authentication:** Managed via Amazon Cognito (JWT).
- **Core Resources:** 12 endpoints covering transaction management, Plaid integration, and budget metadata.
- **Architecture:** Serverless AWS Lambda backend with DynamoDB.

## How to use
You can import the `openapi.yaml` file into [Postman](https://www.postman.com/) or [Swagger Editor](https://editor.swagger.io/) to view the full interactive documentation.
